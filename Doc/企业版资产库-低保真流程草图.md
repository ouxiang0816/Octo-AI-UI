## 6. 四条关键链路

### 平台资产检索复用

```text
进入资产库 -> 平台资产 -> 搜索/筛选 -> 查看详情 -> 引用到当前会话
-> 会话区显示平台来源 -> AI 生成时读取平台上下文
```

### 个人资产上传并引用

```text
进入资产库 -> 个人资产 -> 上传本地文件/查看云端交付件
-> 自动摘要 -> 引用到当前会话 -> 会话区显示个人来源
```

### 混合引用平台 + 个人资产

```text
平台资产引用 + 个人资产引用 -> 会话区展示来源差异
-> 如有冲突显示软提示 -> 继续生成
```

### 会话中管理引用资产

```text
查看已引用资产 -> 查看详情 / 移除 -> 返回资产库补充引用或替换
```
